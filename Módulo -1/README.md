# ☕ Módulo 01 - Fundamentos de Programação com Java

Este diretório contém meus estudos do **Módulo 01** do curso Entra21.

Neste módulo, o foco é aprender os conceitos fundamentais da programação utilizando Java, criando a base necessária para avançar para conteúdos mais complexos.

---

## 🎯 Objetivo do módulo

- Entender a lógica de programação
- Aprender a sintaxe básica do Java
- Desenvolver raciocínio lógico para resolver problemas
- Criar os primeiros programas

---

- [x] Variáveis e tipos de dados  
- [x] Operadores
